# 📝 AI Grammar Checker (Advanced)

This project is an advanced web-based grammar checker inspired by modern tools like Grammarly. It uses API-based AI along with custom rule-based logic to detect and correct grammatical errors in real time.

## 🚀 Features
- Real-time grammar checking
- Error highlighting inside text
- Auto-correction suggestions
- Speech-to-text input
- Dark mode UI
- Custom grammar rules for improved accuracy

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- LanguageTool API
- Web Speech API

## 💡 Description
The application allows users to type or speak text, analyze grammatical errors, and receive intelligent suggestions. It combines API-based grammar checking with additional custom logic to improve detection of common mistakes and informal sentence structures.
